# pyre-code

A self-hosted ML coding practice platform with 68 hands-on exercises spanning attention mechanisms, RLHF, diffusion models, flow matching, and more — designed for ML engineers building practical intuition by implementing from scratch.

- **Repo:** [whwangovo/pyre-code](https://github.com/whwangovo/pyre-code)
- **Stars:** ~713 (2026-04-15, 6 days after launch)
- **Created:** 2026-04-09
- **Language:** Python

## Why It Matters
- Targets the gap between reading ML papers and being able to implement the core primitives — a known pain point for engineers transitioning into applied AI work.
- 68 curated exercises across modern topics (RLHF, flow matching, diffusion) keeps the curriculum current beyond typical academic course offerings.
- Self-hosted: no account creation, no data leakage, runs locally alongside an LLM coding assistant.

## Key Takeaways
- Practice platform for ML fundamentals — attention, RLHF, diffusion, flow matching.
- 68 exercises; self-hosted Python; no external dependencies required.
- Complements AI coding assistants: a builder can run exercises in a local coding environment while using [[codeburn|codeburn]] to track token spend on those sessions.
