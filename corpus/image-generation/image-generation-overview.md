# Image Generation — Overview

Community activity around AI image generation models, prompt engineering libraries, and evaluation resources.

## The Trending Pattern
- Prompt libraries as a category: the "awesome-list" format is being applied to image models just as it was for LLM prompts, system instructions, and DESIGN.md files.
- GPT-Image-2 attracted dedicated community curation within days of availability, following the same flywheel that boosted DALL-E and Stable Diffusion prompt repositories.
- The prompt-library repo as a standalone artifact signals a maturing community — not just sharing outputs, but systematically cataloguing what works.
