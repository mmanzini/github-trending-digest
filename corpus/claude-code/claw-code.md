# claw-code

Community-built Rust CLI agent harness for Claude (and OpenAI). Run prompts and agent workflows from the terminal via `claw prompt "..."`. Explicitly **not affiliated with Anthropic** — positioned as an open-source alternative to the official [[claude-code-overview|Claude Code]].

- **Repo:** [ultraworkers/claw-code](https://github.com/ultraworkers/claw-code)
- **Stars:** 187.6K (as of 2026-04-23), reportedly the **fastest repo in GitHub history to surpass 100K stars** — hit it in under three weeks of its 2026-03-31 launch.
- **Language:** Rust
- **Built on:** oh-my-codex

## Why It Matters
- Signals strong developer appetite for **open agent infrastructure** independent of vendor tooling.
- Demonstrates that a Rust CLI harness can win community mindshare against polished first-party tools.
- The "official vs. open-source" tension here is a recurring story arc — see related [[agent-frameworks-overview]].

## Key Takeaways
- Fastest-to-100K-stars phenomenon — pure content gold for AI-ecosystem creators.
- Built on oh-my-codex; multi-LLM support (Claude + OpenAI).
- A community signal: developers want lightweight, hackable agent CLIs.
