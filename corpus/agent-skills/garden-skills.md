# Garden Skills

**Repo:** [ConardLi/garden-skills](https://github.com/ConardLi/garden-skills)
**Language:** JavaScript | **First seen:** 2026-04-26 | **Peak stars (7-day):** ~1,434

ConardLi's open-source multi-skill collection for Claude Code and compatible agents. Rather than shipping one tightly scoped capability, garden-skills is a curated bundle — a "garden" of single-purpose skills maintained together.

## Included Skills (as of Apr 2026)
- Web design — UI polishing skill inspired by Claude Design
- Knowledge retrieval (RAG) — query and retrieve from indexed sources
- Image generation — GPT-Image-2 prompt workflows
- Additional utilities under active development

## Positioning
Bundles multiple skills where most repos in this cohort ship exactly one. This makes it a reference collection as well as a usable skill pack — similar in spirit to [[awesome-design-md]] for design systems, but scoped to agent skills.

## Key Takeaways
- Multi-skill repo format is a rare pattern; most trending skills are single-purpose
- The RAG skill signals that skill authors are reaching beyond generation toward retrieval
- GPT-Image-2 integration makes this a cross-model skill collection, not Claude-only
- Companion context: [[guizang-ppt-skill]], [[huashu-design]], [[kami]] for the design-skill cohort
