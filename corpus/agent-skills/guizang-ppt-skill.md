# Guizang PPT Skill

**Repo:** [op7418/guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill)
**Language:** HTML | **First seen:** 2026-04-25 | **Peak stars (7-day):** ~3,156

A [[agent-skills-overview|Claude Code Skill]] that converts text prompts into horizontal-swipe, magazine-style HTML slide decks. Outputs a single self-contained HTML file with no external dependencies.

## Features
- 10 distinct slide layouts
- 5 curated visual themes
- WebGL hero backgrounds for cinematic section headers
- Single-file output — the entire deck is one `.html` file
- Part of the April 2026 wave of design-output skills alongside [[kami]], [[huashu-design]], and [[html-ppt-skill]]

## Positioning
Reached 3,000+ stars within four days of creation (2026-04-23), making it one of the fastest-rising design skills in this period. Distinct from [[html-ppt-skill]] in its magazine / horizontal-swipe metaphor and WebGL polish.

## Key Takeaways
- Horizontal-swipe magazine format differentiates it from the vertical-slide norm of [[html-ppt-skill]]
- WebGL backgrounds lower the visual floor for agent-generated presentations significantly
- Single-file output is a deliberate portability constraint — no build step, trivially shareable
