# GitHub Trending – 2026-06-29 (Monday)

## Editor's Take
Two repos from the 30-day list deserve immediate attention. **Ponytail** (67k stars) is a Claude Code plugin built around YAGNI — it plugs directly into Claude Code, Codex, Copilot CLI, and 13 other agents, enforcing a "reuse before you write" ladder that reportedly cuts code volume by ~54% and costs by ~20%. That's a concrete, benchmark-backed tool your audience is building with right now. **Odysseus** (79k stars) is a self-hosted AI workspace that bundles chat, agents, multi-step web research, document editing, email triage, and local model support under one roof — the kind of privacy-first, all-in-one alternative to commercial AI products that routinely drives discussion. From the weekly list, **awesome-evals** is a tightly curated agent evaluation resource worth bookmarking for an eval-focused episode.
