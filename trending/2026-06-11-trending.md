# GitHub Trending – 2026-06-11 (Thursday)

## Editor's Take
Two repos stand out this week. **shadcn/improve** — from the creator of shadcn/ui — distils a pattern that's very much in the air right now: let your most capable model do strategic auditing and write a structured execution plan, then hand that plan to a cheaper model for implementation. It's a concrete, demonstrable workflow for explaining cost-effective LLM orchestration to an AI-agent audience. The second is **JimLiu/baoyu-design**, which packages Claude Design as a local agent skill runnable inside Claude Code and Cursor — a directly practical tool for the creator's own workflow and a good showcase of how Claude Code skills can unlock cloud-only features locally.
