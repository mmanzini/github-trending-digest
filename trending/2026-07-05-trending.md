# GitHub Trending – 2026-07-05 (Sunday)

## Editor's Take
Two repos stand out this week. **Kulaxyz/token-diet** is the most directly actionable find for Claude Code users: it claims a ~31% token-cost reduction by installing an always-on efficiency skill across Claude Code, Codex, Cursor, Windsurf, and Cline with no reported loss of correctness — a single install that applies to every coding agent you use. If the benchmark holds up, that's a compelling "save money on your AI tools" angle. **elder-plinius/T3MP3ST** is the week's top new repo by raw stars: a TypeScript multi-agent platform that orchestrates specialized offensive-security agents against a target, each owning a different attack surface. It's a rare public, production-grade example of multi-agent coordination applied to a real domain and worth dissecting for an "agents working as a team" piece.
