# GitHub Trending – 2026-07-01 (Wednesday)

## Editor's Take
Two repos stand out this week and both belong directly in the creator's feed. **DietrichGebert/ponytail** is the bigger story: 70K stars in under three weeks for a Claude Code plugin that enforces YAGNI at the agent level — it literally makes the agent refuse to write code it doesn't need, and the community momentum is extraordinary for something this opinionated. **Kulaxyz/self-learning-skills** is quieter at 739 stars but arguably more immediately actionable: it's a Claude Code / Cursor skill that watches a session, recognizes a hard-won "golden path," and distills it into a reusable rule for next time — essentially giving your agent institutional memory. Both repos tell a coherent story about where AI coding tooling is heading: less raw capability, more discipline about when *not* to act.
