# GitHub Trending – 2026-05-29 (Friday)

## Editor's Take

Two repos this week are squarely worth a dedicated piece. **`withkynam/vibecode-pro-max-kit`** is a spec-driven coding harness built explicitly for Claude Code, shipping with 12 agents and 32 skills for end-to-end feature development — it's a ready-made example of how teams are extending Claude Code beyond single-shot prompts into structured agentic pipelines. **`Michaelliv/pi-dynamic-workflows`** is the more technically interesting one: it lets Claude itself author JavaScript workflow scripts that fan work out across isolated subagents in parallel, synthesize the results, and report back — a live demo of dynamic multi-agent orchestration that maps directly onto what Anthropic has been building into Claude Code. Both are strong angles for content about the maturing Claude Code ecosystem.
