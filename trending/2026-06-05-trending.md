# GitHub Trending – 2026-06-05 (Friday)

## Editor's Take
Two repos are worth your attention this week. **odysseus** is the rare breakout: 55K+ stars in under a week for a self-hosted AI workspace that bundles agents, MCP tool calling, local LLM support (Ollama, vLLM, llama.cpp), and a vector-based memory layer—essentially a privacy-respecting alternative to ChatGPT and Claude's web interfaces. The breadth and polish here make it strong content for any audience curious about running capable AI locally. The other standout is **memory-os**, a 7-layer agent memory OS built on Qdrant and RAG, which addresses one of the most-asked questions in the agent space right now: how do you give an agent persistent, structured memory? Both projects are actively relevant to your niche and speak to the current wave of local-first, agent-centric tooling.
