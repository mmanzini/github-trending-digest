# GitHub Trending – 2026-05-13 (Wednesday)

## Editor's Take
Two repositories from the monthly list demand attention. **nexu-io/open-design** (39k stars in ~two weeks) is the headline: a local-first, open-source design tool that treats Claude Code Skills as a first-class primitive and auto-detects 16 coding-agent CLIs—Claude Code, Cursor Agent, Gemini CLI and more—letting users swap agents with one click. It's a concrete example of the Claude skill/tool paradigm spreading from the terminal into mainstream creative workflows, and a natural hook for content about where agentic design tooling is heading. **browser-use/browser-harness** is also worth a deep look: its self-healing loop has the LLM write missing browser helper code at runtime, so the harness literally gets smarter every run—a compelling demonstration of the "self-improving agent" pattern that's hard to illustrate better than with a live demo. On the weekly side, **Clawdmeter**—an ESP32 desk widget with animated pixel-art Clawd that reacts to your session token usage—is the kind of hardware-meets-Claude project that performs well as shareable content.
