# GitHub Trending – 2026-05-15 (Friday)

## Editor's Take
Two repos this week are worth putting on the radar. **nexu-io/html-anything** (1,943 stars in under a week) is a locally-running agentic HTML editor that leans on Claude directly to write and ship HTML — it's a clean, concrete example of the "AI agent as IDE" pattern that makes for great demo content. **HermannBjorgvin/Clawdmeter** is a delightful hardware side project: an ESP32 desk dashboard that visualises Claude Code usage in real time, which speaks directly to the Claude Code community and would make compelling short-form content. Neither is a framework or paper — both are finished, shippable things people are already using.
