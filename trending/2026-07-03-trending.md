# GitHub Trending – 2026-07-03 (Friday)

## Editor's Take
Two repos this week are worth your full attention. **DietrichGebert/ponytail** is the clear standout of the month — 73k stars in three weeks for a Claude Code / Cursor plugin that codifies YAGNI for AI agents, forcing them to take the laziest correct path rather than over-engineering solutions. The timing is perfect for content: it's a philosophy-meets-tooling story with mass appeal. Right behind it, **Kulaxyz/self-learning-skills** is a quieter but fascinating idea — a skill that watches a Claude Code session, recognises a golden path the user hard-won, and harvests it into a reusable rule for next time. That "agents that improve their own playbooks" angle hasn't been widely covered yet.
