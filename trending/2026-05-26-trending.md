# GitHub Trending – 2026-05-26 (Tuesday)

## Editor's Take
Two repos this week are directly content-ready for an AI/agent audience: **open-gsd/get-shit-done-redux** and **thananon/9arm-skills**. GSD tackles one of the most practical pain points in Claude Code workflows—context window degradation across long sessions—by orchestrating multi-agent handoffs with persistent planning artifacts and a six-step loop that enforces verification before shipping. 9arm-skills complements it perfectly: a growing library of modular, installable skills for Claude Code covering debugging, code review, and team communication. Together they represent a maturing ecosystem of community tooling built *around* Claude Code, which is strong signal for where agent-assisted development is heading. Worth covering both in the same piece.
