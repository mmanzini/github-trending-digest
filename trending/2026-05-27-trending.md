# GitHub Trending – 2026-05-27 (Wednesday)

## Editor's Take
Two repos this week are genuinely worth a deep dive. **open-gsd/get-shit-done-redux** tackles "context rot" — the quality collapse that happens as an AI agent's context window fills during long coding sessions — with a structured meta-prompting and subagent orchestration system built specifically for Claude Code workflows; it's rare to see context engineering codified this cleanly and it maps directly to what your audience struggles with daily. **UditAkhourii/adhd** is a tighter but equally interesting find: a tree-of-thought reasoning engine with pruning, built on the Claude Agent SDK, that fans out divergent cognitive frames in parallel before scoring and collapsing them — a practical SDK pattern that's worth dissecting on camera. Honorable mention goes to **MoonshotAI/kimi-code**, Moonshot AI's direct play in the coding-agent CLI space with native video input and parallel subagents; it's worth tracking as a Claude Code competitor.
